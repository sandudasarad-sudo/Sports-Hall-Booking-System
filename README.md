# Sports Hall Booking System

## Overview
The Sports Hall Booking System is a ServiceNow application developed as part of the SkillWallet ServiceNow Project.

It enables users to reserve sports facilities through a Record Producer and automates the approval process using Flow Designer.

## Features
- Sports Hall Reservation
- Record Producer
- Reservation Table
- Sports Hall Facility Table
- Approval Workflow using Flow Designer
- Access Controls and Roles

## Technologies Used
- ServiceNow Studio
- Flow Designer
- Record Producer
- Tables
- Access Controls

## Project Components
- Reservation Table
- Sports Hall Facility Table
- Book a Sports Hall Record Producer
- Sports Hall Booking Approval Flow
